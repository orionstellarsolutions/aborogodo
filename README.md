# 🌴 Borogodó — Iconic Brands

"A marca sem vender é projeto de arte, e vender por vender não dura no mercado."
— Manifesto Borogodó.

Uma landing page institucional de altíssimo padrão, desenvolvida sob medida para a Borogodó. Este projeto foi concebido para substituir uma estrutura pesada e lenta em WordPress/Elementor por uma arquitetura em HTML5 puro, gerando uma experiência de marca instantânea, fluida e com carregamento em menos de 0.5 segundos.

Este projeto foi assinado e desenvolvido pela Orion Stellar Solutions (2026).

## ✨ Características do Projeto

### 🎨 1. Estética Bossa & Brisa (Direcionamento Criativo)

Tipografia de Alta Costura: Utilização precisa da fonte Fraunces (emulando as curvas expressivas da Tan Garland) combinada com a precisão geométrica e limpa da Plus Jakarta Sans (emulando a Canva Sans).

Paleta de Cores Identitária: Uma transição harmônica de cores inspiradas nas regionalidades brasileiras (Verde Floresta, Laranja Vibrante, Azul Maresia, Vinho Terracota e Ouro Quente).

### ✍️ 2. Ilustrações Interativas em Linha Contínua (Fine Line Art)

Todas as ilustrações do Hero foram criadas em vetores puros (SVG) com a estética refinada de tatuagens de traço fino, reagindo dinamicamente ao mouse (Parallax) e ao hover dos rituais:

Maresia & Brisa: Ondas marinhas matemáticas que oscilam continuamente e coqueiros que se curvam suavemente sob o vento.

Drink na Praia: Um copo geométrico com líquido dinâmico. O líquido oscila usando física de mola no hover, mas é mantido por um <clipPath> interno para nunca transbordar das paredes do copo.

Sabor de Herança: Uma panela tradicional de barro onde uma colher de madeira mexe em movimento orbital, enquanto os ingredientes emergem e submergem do caldo espesso.

Conversa & Café: Uma xícara de perfil com múltiplas camadas de fumaça vaporosa filtradas por desfoque suave (feGaussianBlur) que se desviam e reagem à brisa do cursor do usuário.

### 🔐 3. Estúdio Borogodó (O Mini Painel Admin Embutido)

Para libertar o projeto da lentidão e das falhas de segurança do WordPress, desenvolvemos um micro-CMS estático integrado diretamente no arquivo HTML:

Acesso Secreto: Ao clicar 5 vezes seguidas no copyright do rodapé ou pressionar o atalho de teclado Ctrl + Shift + E, um painel administrativo oculto desliza na tela.

Edição Visual (Inline): Após autenticar-se com a chave de acesso corporativa (borogodo2026), o site entra em modo editor. O usuário pode clicar diretamente em qualquer texto e alterá-lo na hora (contenteditable).

Compilação e Exportação Limpa: O painel conta com uma função que limpa os marcadores de edição e exporta um arquivo index.html 100% atualizado e pronto para publicação direta no servidor de hospedagem.

### 📰 4. Conexão Nativa com o Substack

Área dedicada para a newsletter oficial "Diários de Borogodó".

Link dinâmico editável diretamente pelo painel administrativo para que o cliente mantenha a ponte direta com sua audiência de forma simplificada.

### 🛠️ Stack Tecnológica

HTML5 & Vanilla Javascript (ES6+)

Tailwind CSS (via CDN) para uma estilização utilitária e ágil.

Google Fonts (Fraunces & Plus Jakarta Sans).

FontAwesome 6.4.0 para iconografia de suporte.

Filtros SVG & CSS Transitions para os efeitos de calor e física fluida das animações.

## 🚀 Como Publicar (Adeus, WordPress!)

Como o projeto é composto por um único arquivo estático de alto desempenho, a publicação é extremamente simples:

Renomeie o arquivo gerado de borogodo_landing_page.html para index.html.

Acesse o gerenciador de arquivos da hospedagem do cliente (ex: cPanel, Hostinger, Locaweb) ou utilize um cliente FTP.

Vá até a pasta raiz do site (geralmente /public_html ou /www).

Remova ou faça backup dos arquivos antigos do WordPress (como wp-content, wp-admin, index.php).

Faça o upload do arquivo index.html diretamente na pasta raiz.

Pronto! O site estará no ar rodando com performance máxima.

## 🔑 Como Utilizar o Painel "Estúdio Borogodó"

Acesse o site publicado no navegador.

Role até o rodapé e clique 5 vezes seguidas sobre o texto de direitos autorais (© 2026 Todos os direitos reservados.).

Alternativamente, utilize o atalho de teclado: Ctrl + Shift + E.

Digite a senha padrão: borogodo2026 e clique em Entrar.

Edite qualquer texto do site clicando diretamente sobre ele.

Para salvar, abra o painel novamente e clique em "Exportar index.html Atualizado".

Substitua o arquivo antigo no servidor pelo novo arquivo baixado.
